# Infrastructure as Code

## Setup

1. Initialize Terraform:
   ```bash
   terraform init
   ```
2. Validate your configuration:
   ```bash
   terraform validate
   ```

## Commands

- Plan your changes:
  ```bash
  terraform plan
  ```
- Apply the infrastructure:
  ```bash
  terraform apply
  ```
- Tear down resources:
  ```bash
  terraform destroy
  ```
